# 🖱️ Melt Mouse

> "Knowing *what* to ask is as important as knowing how to code it."

## What is this?

Melt Mouse is a website built **100% using Artificial Intelligence tools** — no manual coding from scratch. Every layout decision, every line of CSS, every interaction was prompted, directed, and refined through AI collaboration.

This project isn't about replacing developers. It's about understanding what happens when a developer uses AI as a power tool instead of a shortcut.

## Why I built it

I wanted to answer a real question:
**How far can a developer go using only AI — and what does it teach you about the craft?**

The answer: further than expected. And it teaches you a lot about clarity of thought, precision in communication, and how good requirements produce good software.

## What it demonstrates

- Prompt engineering as a technical skill
- AI-assisted UI/UX design and layout
- Iterative refinement through natural language
- Understanding output well enough to direct and correct it

## Tech involved

| Tool | Purpose |
|------|---------|
| AI (Claude / ChatGPT) | Code generation & design |
| HTML5 | Structure |
| CSS3 | Styling & layout |
| JavaScript | Interactions |

## Key takeaway

The developer who understands AI deeply becomes 10x more productive than the one who ignores it — or blindly depends on it.
